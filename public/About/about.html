About the Data

Data Sets


The Data Sets used throughout this project include the following:
  • 2023-2024 New York Knicks Players' Game Logs
  • 2023-2024 NBA Teams' Game Logs
  • 2023-2024 NBA Player Averages
  • 2023-2024 NBA Team Averages

These data sets are provivded through the data collections which are publically available at Basketball Reference by Stathead. They are taken by Basketball Reference from Sportradar, who are the official stats partner of the NBA.

Data Collection

The data was collected directly via Basketball Reference. The site allows users to download .CSV files, which I then uploaded to a mySQL database such that they are accessible va the web. mySQL Server and mySQL Workbench were used to set up this database.




About the Analysis

Methods


Data analysis was done mainly using MatLab.

Temporary views were created by executing MySQL statements in my database. Afterwards, those temporary views were downloaded and imported into MatLab. There, the data was plotted and I conducted linear regression analyses to analyze the relationships present.

Potential for Error

There is potential for error from a variety of sources, including:
  • Incorrect stat-tracking by official stat trackers
  • Improper or unanticipated rounding of floating point values by MySQL Server
  • User error in data migration or analysis




About the Site

Front-End


The site's front-end is a custom built webpage that uses a few third party libraries for GUI support

The site is comprised exclusively of HTML, CSS, and JavaScript, with the CDN DataTables library used for table visualization

Back-End

This site is a Node.JS site and uses a few node modules on the back-end, including:
  • Express, for web-server capabilities
  • MySQL2, for accessing a remote MySQL database
The MySQL database is hosted on an AWS RDBMS instance
