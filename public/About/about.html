About the Data

Data Sets

The data sets which are used in this project include the following:

  • The 2023-2024 New York Knicks Player Game Logs and Averages
  • The 2023-2024 NBA Player Averages
  • The 2023-2024 NBA Team Stats
  • Public Opinion

These data sets are provivded through the data collections which are publically available at Basketball Reference by Stathead. They are taken by Basketball Reference from Sportradar, who are the official stats partner of the NBA. The Public Opinion dataset was collected through social media polling on Knicks and NBA fan forums.

Data Collection

The data was collected directly via Basketball Reference. The site allows users to download .CSV files, which I then uploaded to a mySQL database such that they are accessible va the web. mySQL Server and mySQL Workbench were used to set up this database.
