This page contains data visualizations made using MatLab. The data was extracted from the mySQL database using mySQL Workbench and analyzed using MatLab through data visualization via graph creation and analysis of statistical significance. Each graph below will have an associated description as well as a brief rundown of values that relate to statistical significance.



This graph contains teams organized by wins and number of double digit scorers

  • A linear regression model was performed and a line of best fit was added to the data
  • While the R squared value is relatively low, a low p-value indicates that there is a statistical significance in this relationship
  • These findings suggest that teams with fewer double digit scorers perform to a higher standard during the NBA regular season
  • Some other factors that could impact this relationship include team chemistry, injury continuity, and age
