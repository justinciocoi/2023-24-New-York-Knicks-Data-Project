Initial Intent

The initial intent of this project was to accomplish the following goals:
  • Create a streamlined database for accessing stats specifically for the 2023-24 New York Knicks
  • Host this data using a mySQL database which is accessible from the web
  • Create some statistical analysis based on my predictions as an avid NBA watcher

Evolution of the Project

As the project progressed, a few more ideas emerged
  • Including logs and average stats for other teams such that they can be compared to eachother
  • Use Node.JS to create a web-page that acts as a GUI for my created database and related details
  • Use MatLab to create data visualzation for temporary SQL views that add statistical information
